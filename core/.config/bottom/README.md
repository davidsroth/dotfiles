# bottom

A customisable, graphical process/system monitor for the terminal written in Rust.

- **Docs**: <https://clementtsang.github.io/bottom/stable/>
- **Repo**: <https://github.com/ClementTsang/bottom>

## Local customizations

`bottom.toml` is mostly stock (all values are commented out). It serves as a reference for available options and a hook for future tweaks to layout, theme, or widget defaults. The config currently defers to `btm` built-in defaults.
