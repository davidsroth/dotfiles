# Espanso

A cross-platform text expander written in Rust.

- **Docs**: <https://espanso.org/docs/>
- **Repo**: <https://github.com/espanso/espanso>

## Local customizations

`config/default.yml` sets `search_shortcut: ALT+SHIFT+SPACE` so the searchable trigger overlay is available on-demand. Match definitions live under `match/` and are machine-specific (not stowed).
