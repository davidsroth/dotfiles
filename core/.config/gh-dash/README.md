# gh-dash

A GitHub PR/issue dashboard for the terminal, built on top of `gh`.

- **Docs**: <https://dlvhdr.github.io/gh-dash/>
- **Repo**: <https://github.com/dlvhdr/gh-dash>

## Local customizations

`config.yml` defines several PR sections (Review Requested, Recently Updated, Tower & Nectar, My Pull Requests, Assigned to Me) and a matching set for Issues. Keybindings attach tmux and lazygit workflows to PR lists (checkout, diff, open in browser, copy URL, launch lazygit in a new window or popup). A custom theme uses Catppuccin Mocha-ish accent colors.
