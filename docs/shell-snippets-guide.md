# Shell Snippets Guide

This document maps the modular shell configuration used in this dotfiles repo.

## Files

- **`core/.config/shell/aliases.sh`** — Command aliases organized by category (core replacements, Git, Docker, system management, development tools, utilities, platform-specific, and safety aliases). Sourced from `.zshrc`.
- **`core/.config/shell/functions.sh`** — Shell functions organized by category (date/time utilities, temporary file management, logging and note-taking, clipboard helpers, file operations, directory navigation, Git utilities, system information, cleanup, and process utilities). Sourced from `.zshrc`.

For details on what is defined in each file, open the files directly or use `grep` to locate specific aliases and functions.
